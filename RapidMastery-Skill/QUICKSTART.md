# Rapid Mastery - 快速上手指南

## 一分钟入门

```bash
# 格式: /rapid-mastery <主题> [深度] [时间]

# 快速了解一个概念
/rapid-mastery 区块链 know 1h

# 学会应用一项技术（最常用）
/rapid-mastery Python装饰器 apply 1day

# 深度精通一个领域
/rapid-mastery 机器学习 master 1week
```

---

## 深度说明

| 深度 | 适合场景 | 学习时长 | 包含内容 |
|------|----------|----------|----------|
| **know** | 了解概念、聊天谈资 | 1-2小时 | 概念解构 + 类比 + 简要测试 |
| **apply** | 工作中使用、写作业 | 半天-3天 | 完整7步流程 |
| **teach** | 教别人、写博客 | 1周 | 完整流程 + 教学演练 |
| **master** | 成为专家、面试 | 2周+ | 完整流程 + 源码 + 复杂项目 |

## 交互速查

| 你想做什么 | 说什么 |
|-----------|--------|
| 跳过当前步骤 | "跳过这步" 或 "直接到测试" |
| 重新解释概念 | "用另一个类比解释" |
| 增加难度 | "给我更难的题目" |
| 降低难度 | "简单点，我是新手" |
| 导出复习计划 | "导出到日历/待办" |
| 复习旧主题 | "/rapid-mastery --review 主题名" |
| 查看进度 | "我现在在哪一步？" |

## 使用技巧

### 1. 善用类比
如果系统给的类比不理解，直接说：
```
"换一个类比，用[生活/编程/游戏]的角度"
```

### 2. 真实反馈
费曼环节被追问时，不要装懂：
```
"其实我这里不太明白..."
```
→ 系统会针对性补充解释

### 3. 项目定制
如果觉得项目不合适：
```
"换一个项目，偏向[数据分析/Web开发/算法]方向"
```

### 4. 组合学习
学习相关主题时：
```
/rapid-mastery Python基础 apply 3day
/rapid-mastery Python装饰器 apply 1day
/rapid-mastery 设计模式 apply 1week
```

## 常见问题

**Q: 学习过程中断了怎么办？**
A: 直接说"继续"或"回到第X步"，系统会恢复上下文。

**Q: 可以学习非技术主题吗？**
A: 可以！如 `/rapid-mastery 宏观经济 apply 1week`

**Q: 学习时间不够？**
A: 缩小范围：`/rapid-mastery "Python装饰器之缓存实现" apply 2h`
